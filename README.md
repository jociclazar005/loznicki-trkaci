# loznicki-trkaci

Sarađujem sa sportskim udruženjem "Loznički trkači" još od njihovog osnivanja, te sam bio volonter na njihovoj prvoj trci.

Na svim trkama koje organizuje ova organizacija sam redovan volonter, a u sklopu Lozničke trkačke lige sam čak radio i posao merenja vremena.

Za svoj maturski rad u srednjoj školi napravio sam sajt loznickitrkaci.com
